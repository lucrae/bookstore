Djuliana Imperio (dimperiodom)

1. Register Account

PageRegister.java This is a form that the user can fill out when making a new account. All fields are required. On submit, it will go to RegisterUser. It has the option to unhide the password.
RegisterUser.java Makes sure that the email does not exist, and that the passwords coincides. If these are complied, a new user is made, otherwise it will give an error message and will redirect back to the Register Page.

2. Log In (Existing Account)

PageLogin.java The user can log in from this page, and it will be verified by the CheckLogin servlet.
CheckLogin.java If the email exists, and the password matches the email, it will proceed to the bookstore, otherwise an error will be given.

3. Browse the bookstore

PageMain.java This is the main browsing web page, that allows the user to see what books are available in the bookstore. It allows the user to filter by author/title, genre, and order it by year published or average rating.
FilterBooks.java This is similar to the main page, except it displays the filter criteria that the user had previously made ath the bottom part.

Internal functions

registerUser This function allows new users to create an account
getUserByEmail This functions gets the users information from the database based on the email that is registered
loginUser This function is to check wether the email matches the password that has been input when the user tries to log in
getBookList This function gets all registered books from the database and displays them in a list.
buyBook This function reduces the stock buy one when a user buys a book
getLatestBooks This function gets the latest books registered in the database and displays them to the user
getBookRating Given a book, this returns the average of all the ratings made for this book
filterCriteria This function calls the proper function (sql query) based on the user input, and returns the same set of books that the called on function also returned. It basically redirects which function to call:
  • getBySearch
  • getByGenre
  • getBySort
  • getByRatingDesc
  • getByGenreAndSearch
  • getByGenreAndSort
  • getBySearchAndSort
  • getByGenreAndRating
  • getBySearchAndRating
  • getByAll
  • getByAllRating

ReviewUpdate.java From the Accounts/Settings page, this allows the user to edit their previous reviews in case they change their mind

Internal functions

getReviews This return all the reviews of a certain book
