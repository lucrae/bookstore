registerUser
Description: This function allows new users to create an account.
Use: The user enters this page if they have no account and clicks on "Create an account".
loginUser
Description: This function allows registered users to log in to the webpage.
Use: The user enters this page if they have an account and clicks on "Log in".
listBooks
Description: This function gets all registered books from the database and displays them in a list.
Use: The user and admin both use this function in their respective webpages. For the users it is in the main page after logging in. For the admin it is visible when clicking on "Books".
newestBooks
Description: This function gets the latest books registered in the database and displays them to the user.
Use: The users can view this in the main page after logging in.
searchBooks
Description: This function allows the user to search the database for different books. Either free text or by filtering.
Use: The user can use this function from the main page after logging in.
getReviews
Description: This function allows the user to view previous reviews made by other users.
Use: The user sees these reviews when viewing the info of the book after clicking on its title.

getBookInfo
Description: This function allows the user to view the information of books registered in the database.
Use: The user enters this page from the page with the list of books. When the title of the book is pressed, the user gets the information of the book.
usersBooks
Description: This function allows the user to view the books previously purchased by them.
Use: The user enters this page when they go to their account.
addUserReview
Description: This function allows the user to add a review to books previously purchased by them.
Use: The user is able to do this when in the account page.
getUserReview
Description: This function allows the user to view previous reviews made by them.
Use: The user sees these reviews when in the account page and viewing books previously purchased by the user.
editUserReview
Description: This function allows the user to edit previous reviews made by them.
Use: The user can edit reviews when viewing previous purchases made by them.
editUserInfo
Description: This function allows the user to edit their information, such as address or e-mail.
Use: The user enters this page when clicking on "Settings" in the "Account" page.
updateUserInfo
Description: This function allows the user to save new/changed information to the database.
Use: The user enters this page when clicking on "Settings" in the "Account" page.

editUser
Description: This function allows the admin to edit users that are registered in the database
Use: The admin enters this page from the page with the list of users. When "Edit" is pressed, the admin gets the option to edit information of the user.
deleteUser
Description: This function allows the admin to delete users that are registered in the database
Use: The admin enters this page from the page with the list of users. When "Edit" is pressed, the admin gets the option to delete the user from the database.
updateUser
Description: This function allows the admin to edit users that are registered in the database
Use: The admin enters this page from the page with the list of users. When "Edit" is pressed, the admin gets the option to edit information of the user and then update the information in the database.
listUsers
Description: This function allows the admin to view a list of all the users registered in the database
Use: The admin enters this page from the admin home page. When "Users" is pressed, the admin gets all registered users in a list.
searchUser
Description: This function allows the admin to search for users that are registered in the database
Use: The admin can use this function on the page with the list of users.
addBook
Description: This function allows the admin to add new books to the database
Use: The admin enters this page from the page with the list of books. When "Add new book" is pressed, the admin can fill in information about a new book.
editBook
Description: This function allows the admin to edit information of a book.
Use: The admin enters this page from the page with the list of books. When "Edit" is pressed, the admin gets the option to edit information of the book.
