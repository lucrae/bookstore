RegisterUser.java Makes sure that the email does not exist, and that the passwords coincides. If these are complied, a new user is made, otherwise it will give an error message and will redirect back to the Register Page.
PageRegister.java This is a form that the user can fill out when making a new account. All fields are required. On submit, it will go to RegisterUser.
PageLogin.java The user can log in from this page, and it will be verified by the CheckLogin servlet.
PageMain.java This is the main browsing web page, that allows the user to see what books are available in the bookstore. It allows the user to filter by author/title, genre, and order it by year published or average rating.
FilterBooks.java This is similar to the main page, except it displays the filter criteria that the user had previously made ath the bottom part.
CheckLogin.java If the email exists, and the password matches the email, it will proceed to the bookstore, otherwise an error will be given.
